# MarkDown

## 语法

### 标题

标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可

```
# 一级标题

## 二级标题

### 三级标题
```

### 列表

在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 1. 1. 符号要和文字之间加上一个字符的空格

1. 无序列表
1. 有序列表

```
* a
* b
* c

1. a
1. b
1. c
```
