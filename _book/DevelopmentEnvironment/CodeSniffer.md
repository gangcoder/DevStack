## PHP_CodeSniffer

PHP5写的用来检查嗅探PHP代码是否有违反一组预先设置好的编码标准的一个包

### 安装

```
sudo pear install PHP_CodeSniffer
```

### 使用

```
phpcs /path/to/code/yourfile.php
```
